1. **pip 버전을 업그레이드 합니다. <code>python.exe -m pip install --upgrade pip</code>**
2. **requirements 폴더로 이동합니다. <code>pip install -r requirements.txt</code>**
3. **필요한 패키지를 설치합니다. <code>pip install -r requirements.txt</code>**
